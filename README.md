# java-finalweb-backup

#### 介绍
javaee网页的备份【软件课程设计的备份】

#### 软件架构
使用jsp编写了前端页面；
编写了后端的servlet；
使用jdbc连接数据库；
【warning】数据库需要MySQL支持

#### 安装教程
【attention】关于MySQL，使用navicat可以快速建立数据库
【attention】需要使用tomcat
【attention】需要使用idea

将项目直接加载至idea即可使用
